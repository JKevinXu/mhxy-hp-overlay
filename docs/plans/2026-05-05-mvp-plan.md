# MVP Plan: 可见血量显示工具

## Task 1: Repo scaffold

- README with boundaries.
- pyproject package.
- src package and tests.
- GitHub Actions CI.

## Task 2: Static image analyzer

- Implement ROI parsing.
- Implement HSV red HP bar detection.
- Generate synthetic test fixture.
- Unit test full/partial/empty HP bars.

## Task 3: CLI

- `mhxy-hp analyze-image` for fixtures/screenshots.
- `mhxy-hp watch-screen` for read-only screen ROI sampling.
- JSON and terminal table output.

## Task 4: Calibration

- Add config file with named ROIs.
- Add command to print mouse/screen coordinate info without clicking.
- Optional debug mask image output.

## Task 5: UI

- Terminal dashboard first.
- Later optional transparent overlay window, still no game control.
